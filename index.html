Buat Laporan / Saran Baru Lihat Semua Laporan

Bagaimana Cara Kerjanya?

  1. Klik tombol "Buat Laporan / Saran Baru".
  2. Anda akan diarahkan ke halaman GitHub Issues.
  3. Tulis judul dan isi laporan Anda dengan jelas. (Anda mungkin perlu login ke GitHub).
  4. Tim kami akan meninjau dan merespons laporan Anda secara transparan.
  5. Anda dapat melihat status semua laporan di tombol "Lihat Semua Laporan".
